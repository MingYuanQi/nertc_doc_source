为满足用户在音视频通话过程录制和备案等需求，音视频通话 2.0 提供云端录制功能，支持一对一或多人的音视频通话过程录制。通过云端录制功能，可以将语音聊天、视频聊天的内容录制并储存下来，以便后续提供给更多人观看回放或存档。

云端录制适用于在线会议、远程教育、在线医疗等行业场景，为内容存档、录制回放等需求提供便捷的录制方案。

## 新版云端录制方案

云端录制服务于 2021 年 11 月 18 日上线适用于 V4.5.0 及之后版本 SDK 的云端录制方案，为您提供录制任务的启停、查询以及录制订阅列表的更新等功能。您可以将云端录制的相关配置完全交由服务端控制，例如订阅类型、录制布局等等，无需配合控制台进行布局设置。

::: note note

- 若您尚未接入过网易云信的云端录制功能，建议直接使用纯服务端配置的[云端录制方案](/docs/jcyOTA0ODM/zE4NDg0NTM)。适用 V4.5.0 及之后版本 SDK 的整套服务端 RESTful API 提供更丰富的录制布局配置、支持订阅黑白名单设置，方案更加细节和灵活。涉及录制画布配置问题，请联系技术支持。
- 若您已经使用过网易云信的云端录制方案，请注意[旧版云端录制](/docs/jcyOTA0ODM/zE1OTE4OTI) API 不再更新迭代，我们推荐您尽快切换至新版云端录制 API，无需客户端作任何改动，且能体验更为灵活的云端录制配置。

:::
