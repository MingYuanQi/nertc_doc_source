耳返即耳机采集监听，在设备上插入耳机或耳麦后，可以从耳机侧听到本设备麦克风采集到的声音，主要用来监听本地采集的音频。耳返音频具备低延时、高音质等特征，可以实时向主播等参与音频互动的成员反馈本端的声音数据，让主播可以实时听到本端的声音。耳返功能一般用于在线 KTV、连麦 PK、演唱会等娱乐场景。

NERTC SDK 支持耳返功能，同时支持调节耳返的音量。

## 功能介绍

耳返功能可以应用于以下场景：

- 连麦 PK 场景中，用户需要实时监听自己的声音。

- 在线 KTV 场景中，用户需要实时监听自己的歌唱声音，以便进行自我调节。

## <span id="注意事项">注意事项</span>

- 开启耳返功能后，必须连接上耳机或耳麦，才能正常使用耳返功能。建议通过 <a href="https://doc.yunxin.163.com/nertc/api-refer/uniapp/typedoc/Latest/zh/html/interfaces/nertccallback.nertccallback-1.html#onaudiodevicechanged" target="_blank">`onAudioDeviceChanged`</a> 回调监听播放设备的变化，当监听到播放设备切换为耳机时才开启耳返。

- 加入房间前或加入房间后均可开启耳返。

## <span id="实现方法">开启耳返</span>

- 调用 <a href="https://doc.yunxin.163.com/nertc/api-refer/uniapp/typedoc/Latest/zh/html/modules/nertc.nertc-1.html#enableearback" target="_blank">`enableEarback`</a> 方法可以开启或关闭耳返功能。

    ::: note note
    耳返功能可以随时开启，但只有当插入耳机或耳麦时，耳返功能才会生效，当拔出耳机或耳麦时会自动暂停，再次插入耳机或耳麦自动恢复，且耳返功能可以实时关闭。
    :::

- 开启耳返功能时，通过 `volume` 参数可以设置采集耳返音量大小，支持设置为 0 ~ 100，默认为 100。

## <span id="设置耳返音量">调节耳返音量</span>

调用 <a href="https://doc.yunxin.163.com/nertc/api-refer/uniapp/typedoc/Latest/zh/html/modules/nertc.nertc-1.html#enableearback" target="_blank">`enableEarback`</a> 方法开启耳返功能之后，您可以调用 <a href="https://doc.yunxin.163.com/nertc/api-refer/uniapp/typedoc/Latest/zh/html/modules/nertc.nertc-1.html#setearbackvolume" target="_blank">`setEarbackVolume`</a> 方法设置耳返音量。

## 示例代码

**示例代码** 如下：

```javascript
//开启耳返，并且将采集音量设置为 100
this.engine.enableEarback({
    enabled: true,
    volume: 100
});

//中途将耳返音量设置为 100
this.engine.setEarbackVolume(100);
```
