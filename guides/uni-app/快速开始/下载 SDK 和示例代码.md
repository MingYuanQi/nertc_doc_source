本文介绍如何下载 NERTC SDK、API Sample 和场景方案源码。

## <span id="SDK 下载">下载 SDK</span>

请通过 [**SDK 下载中心**](https://doc.yunxin.163.com/nertc/sdk-download?platform=uniapp) 下载各平台最新稳定版 SDK 包。

:::note note
根据不同行业、不同场景的不同业务需求，NERTC 提供不同版本的 SDK 包，请在 <a href="https://yunxin.163.com/">官网首页</a> 通过微信、在线消息或电话等方式联系网易云信商务经理，为您推荐针对您的场景最合适、最优的版本。
:::
