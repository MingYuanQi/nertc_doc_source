网易云信音视频通话 Wwise SDK（简称 NERTC Wwise SDK）配套提供示例游戏 Demo。可帮助您了解在 Unreal Engine 工程中如何使用 NERTC Wwise 插件。

## 开发环境要求

- Wwise 示例工程使用 Unreal Engine 4.27 创建。
- WWise 版本为 2022.1.8.8316。

## 操作步骤

1. 将 IntegrationDemo 附带的 Wwise 工程复制到示例游戏的同级目录下。
    - Wwise Project 安装路径：`{Wwise-INSTALL-ROOT}/SDK/samples/plugins/nertc/IntegrationDemo`
    - Demo 安装路径：`{Wwise-INSTALL-ROOT}/SDK/plugins/nertc/Unreal/GameVoiceWwiseDemo`

        ::: note notice
        以上路径说明中，`{Wwise-INSTALL-ROOT}` 为 Wwise 对应版本的安装目录。
        :::

2. 打开 Audiokinetic Launcher 应用，选择左侧 Unreal Engine 标签页后，单击上方的"Open other"按钮，选择 `GameVoiceWwiseDemo` 所在的 `GameVoiceWwiseDemo.uproject` 文件。

3. 选择 Demo 工程，并单击 **Intergate Wwise into Project** 按钮集成到 Unreal 项目中。

    ::: note notice
    集成 Wwise 插件时，您需要选择 IntegrationDemo 中附带的 `NERtcProject.wproj` 项目文件。
    :::

4. 集成成功后，会在 Unreal 工程 **Plugins** 目录下生成 **Wwise** 相应的目录。

5. 使用 Unreal Engine 打开 `GameVoiceWwiseDemo.uproject` 工程，并需要在项目中配置 **WwiseProject** 路径和 **SoundBanks** 产物路径。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/3c0244f1e5f74d20dcaa00f3bb2c76bd/image.png" style="width:80%;border: 1px solid #BFBFBF;">

6. 打开 **Window** > **Wwise Browser** , 单击 **Generate SoundBanks** 生成 **SoundBanks** 资源。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/9fd895ba3f95c567b559afe93d84848c/image.png" style="width:80%;border: 1px solid #BFBFBF;">

7. 完成上述步骤后，可以单击 **Play** 按钮运行游戏。

    <img alt="image.png" src="https://yx-web-nosdn.netease.im/common/5720c02b7862e96d4c351fd1bfa6c984/image.png" style="width:80%;border: 1px solid #BFBFBF;">
