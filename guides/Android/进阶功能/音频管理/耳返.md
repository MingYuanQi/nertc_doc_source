耳返即耳机采集监听，在设备上插入耳机或耳麦后，可以从耳机侧听到本设备麦克风采集到的声音，主要用来监听本地采集的音频。耳返音频具备低延时、高音质等特征，可以实时向主播等参与音频互动的成员反馈本端的声音数据，让主播可以实时听到本端的声音。耳返功能一般用于在线 KTV、连麦 PK、演唱会等娱乐场景。

NERTC SDK 支持耳返功能，同时支持调节耳返的音量。
## 功能介绍
耳返功能可以应用于以下场景：
- 连麦 PK 场景中，用户需要实时监听自己的声音。

- 在线 KTV 场景中，用户需要实时监听自己的歌唱声音，以便进行自我调节。

## <span id="注意事项">注意事项</span>

- 开启耳返功能后，必须连接上耳机或耳麦，才能正常使用耳返功能。建议通过 <a href="https://doc.yunxin.163.com/nertc/api-refer/android/doxygen/Latest/zh/html/interfacecom_1_1netease_1_1lava_1_1nertc_1_1sdk_1_1_n_e_rtc_callback_ex.html#a5142f0ed83fdce1e4c3aa2645dccb320" target="_blank">`onAudioDeviceChanged`</a> 回调监听播放设备的变化，当监听到播放设备切换为耳机时才开启耳返。
- 在 V4.0.0 版本中，<a href="https://doc.yunxin.163.com/nertc/api-refer/android/doxygen/Latest/zh/html/classcom_1_1netease_1_1lava_1_1nertc_1_1sdk_1_1_n_e_rtc_ex.html#ae7930c0e8c9507ee7b2cdd84bbc6cc2c" target="_blank">`enableEarback`</a> 方法的 `volume` 参数无效，请使用 <a href="https://doc.yunxin.163.com/nertc/api-refer/android/doxygen/Latest/zh/html/classcom_1_1netease_1_1lava_1_1nertc_1_1sdk_1_1_n_e_rtc_ex.html#acd692e03b116ff5ec4eafa2949379065" target="_blank">`setEarbackVolume`</a> 接口设置耳返音量。
- 加入房间前或加入房间后均可开启耳返。

## <span id="实现方法">实现方法</span>

- 调用 <a href="https://doc.yunxin.163.com/nertc/api-refer/android/doxygen/Latest/zh/html/classcom_1_1netease_1_1lava_1_1nertc_1_1sdk_1_1_n_e_rtc_ex.html#ae7930c0e8c9507ee7b2cdd84bbc6cc2c" target="_blank">`enableEarback`</a> 方法可以开启或关闭耳返功能。

    ::: note note
    耳返功能可以随时开启，但只有当插入耳机或耳麦时，耳返功能才会生效，当拔出耳机或耳麦时会自动暂停，再次插入耳机或耳麦自动恢复，且耳返功能可以实时关闭。
    :::

- 开启耳返功能时，通过 `volume` 参数可以设置采集耳返音量大小，支持设置为 0 ~ 100，默认为 100。也可以通过 <a href="https://doc.yunxin.163.com/nertc/api-refer/android/doxygen/Latest/zh/html/classcom_1_1netease_1_1lava_1_1nertc_1_1sdk_1_1_n_e_rtc_ex.html#acd692e03b116ff5ec4eafa2949379065" target="_blank">`setEarbackVolume`</a> 方法设置耳返音量，具体请参考<a href="https://doc.yunxin.163.com/nertc/guide/jUzMjAxNTc?platform=android#%E8%AE%BE%E7%BD%AE%E8%80%B3%E8%BF%94%E9%9F%B3%E9%87%8F" target="_blank">设置耳返音量</a>。

## <span id="示例代码">示例代码</span>

```
NERtcEx.getInstance().enableEarback(enable,volume);
```
