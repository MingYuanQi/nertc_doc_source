
## <span id="SDK 下载">SDK 下载</span>

NERTC 的 SDK 包按功能可分为音视频 SDK 和纯音频 SDK，您可以根据需要下载对应的 SDK 包。

- 音视频 SDK：包含音视频完整能力，功能更丰富。
- 纯音频 SDK：只包含音频通话能力，SDK 包的体积会更小。
根据不同行业、不同场景的不同业务需求，NERTC 提供不同版本的 SDK 包，请在<a href="https://yunxin.163.com/">官网首页</a>通过微信、在线消息或电话等方式联系云信商务经理，为您推荐针对您的场景最合适、最优的版本。

各平台最新稳定版 SDK 包的下载地址：[SDK下载中心](https://yunxin.163.com/im-sdk-demo?category=nrtc)。
