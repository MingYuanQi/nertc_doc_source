

耳返即耳机采集监听，在设备上插入耳机或耳麦后，可以从耳机侧听到本设备麦克风采集到的声音，主要用来监听本地采集的音频。耳返音频具备低延时、高音质等特征，可以实时向主播等参与音频互动的成员反馈本端的声音数据，让主播可以实时听到本端的声音。耳返功能一般用于在线 KTV、连麦 PK、演唱会等娱乐场景。

NERTC SDK 支持耳返功能，同时支持调节耳返的音量。
## 功能介绍
耳返功能可以应用于以下场景：
- 连麦 PK 场景中，用户需要实时监听自己的声音。

- 在线 KTV 场景中，用户需要实时监听自己的歌唱声音，以便进行自我调节。

## <span id="注意事项">注意事项</span>

- 耳返功能相关方法的返回值小于 0 时，表示方法调用失败。
- 加入房间前或加入房间后均可开启耳返。
## <span id="实现方法">实现方法</span>

调用 [enableEarback](https://doc.yunxin.163.com/nertc/api-refer/macOS/doxygen/Latest/zh/html/classnertc_1_1_i_rtc_engine_ex.html#a550356dfd28a11cfb1db95f360c13ffa) 可以开启或关闭耳返功能。耳返功能可以随时开启，且在未插入耳机或耳麦时也可生效，但是是从扬声器直接播放耳返音频，因此为了保证耳返质量，**建议在插入耳机或耳麦时使用此功能**。

开启耳返功能时，通过 volume 参数可以设置采集耳返音量大小，支持设置为 0 ~ 100，默认为 100。也可以通过 [setEarbackVolume](https://doc.yunxin.163.com/nertc/api-refer/macOS/doxygen/Latest/zh/html/classnertc_1_1_i_rtc_engine_ex.html#af1fccbd78c318eb5a4b753454f88aa93) 设置耳返音量。

## <span id="示例代码">示例代码</span>

```c++
// 首先设置耳返参数
int res = kNERtcErrFatal;
bool enabled = true;
uint32_t volume = 50; //耳返音量
if (nrtc_engine_)
{
    res = nrtc_engine_->enableEarback(enabled, volume);
}
if (kNERtcNoError == res)
{
    //开启耳返成功
}
else
{
    //开启耳返失败
}

```
