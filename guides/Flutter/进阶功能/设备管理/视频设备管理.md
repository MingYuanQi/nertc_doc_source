<!--keywords:音视频通话,摄像头切换,闪光灯,对焦,曝光 -->

在各种视频的业务场景中，用户经常会需要调整设备的摄像头，例如调整曝光、对焦和闪光灯等功能设置，使被摄物体成像清晰、亮度适宜。 NERTC SDK 提供整套的摄像头管理方法，以便用户灵活切换前置摄像头、后置摄像头，并对摄像头进行缩放、对焦和曝光等相关配置。

## 功能介绍

NERTC SDK 提供相关接口，支持检测拍摄相关设备的能力，例如检测是否支持闪光灯常开等。同时支持多种摄像头相关的设置方法，包括：
- 摄像头相关：
    - 摄像头缩放：支持动态调整缩放比例。
    - 摄像头切换：支持切换或指定前后置摄像头。
- 闪光灯：支持开关闪光灯。
- 对焦：支持手动选择对焦位置，摄像头自动对焦。
- 曝光：支持手动选择曝光区域，摄像头自动曝光。

## 注意事项

- 部分设备可能不支持开启闪光灯或摄像头缩放，直接调用接口调节设备可能会失败。因此用户操作前应先调用相关检测接口，判断当前设备是否支持调整设备的相关设置。
- 请在相机启用后再调用视频设备相关的设置方法，例如在调用 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/startVideoPreview.html" target="_blank">`startVideoPreview`</a> 方法或 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/joinChannel.html" target="_blank">`joinChannel`</a> 方法后。



## 摄像头缩放

### 配置步骤

1. 调用 [`isCameraZoomSupported`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/isCameraZoomSupported.html)，检测设备是否支持摄像头缩放功能。
2. 调用 [`setCameraZoomFactor`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/setCameraZoomFactor.html) 方法设置具体需要的缩放比例，可在通话过程中动态调整。

    ::: note note
    在设置需要的摄像头缩放比例前，您可以调用 [`getCameraMaxZoom`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/getCameraMaxZoom.html) 方法获取设备支持的最大缩放比例或调用 [`getCameraCurrentZoom`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/getCameraCurrentZoom.html) 方法获取摄像头当前的缩放比例。
    :::
    
### 示例代码

```
//检测设备是否支持摄像头缩放
await _engine.deviceManager.isCameraZoomSupported();

//获取设备支持的最大缩放比例
await _engine.deviceManager.getCameraMaxZoom();

// 设置摄像头缩放比例，目前仅支持 Android 平台
int curZoom = await _engine.deviceManager.getCameraCurrentZoom();
await _engine.deviceManager.setCameraZoomFactor(curZoom + 2); //放大
await _engine.deviceManager.setCameraZoomFactor(curZoom - 2); //缩小
```

## 摄像头切换

### 配置步骤

- 调用 [`switchCamera`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/switchCamera.html) 方法在前置摄像头和后置摄像头之间切换，通话过程中也可随时切换。
- 调用 [`switchCameraWithPosition`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/switchCameraWithPosition.html) 方法指定使用前置摄像头（`kNERtcCameraPositionFront`）或后置摄像头（`kNERtcCameraPositionBack`），通话过程中也可随时指定。

::: note note
- 若切换摄像头后的最大分辨率不支持设置的编码参数，SDK 会根据设备实际支持的分辨率，调整编码的实际分辨率和对应的实际码率。
- V5.3.0 及之后版本，关闭再开启本地视频采集后，采集的摄像头行为变更，由之前的延用上一次摄像头配置，改为默认采用[`switchCameraWithPosition`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/switchCameraWithPosition.html) 接口中指定的前置摄像头或后置摄像头。若在房间内重新开启摄像头采集，请调用 [`switchCamera`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/switchCamera.html) 方法重新选择所需要的前置或后置摄像头。

:::

### 示例代码
```
// 切换前后摄像头
int ret = await _engine.deviceManager.switchCamera();
 
//指定前置摄像头
int ret = await _engine.deviceManager.switchCameraWithPosition(NERtcCameraPosition.kNERtcCameraPositionFront);

//指定后置摄像头
int ret = await _engine.deviceManager.switchCameraWithPosition(NERtcCameraPosition.kNERtcCameraPositionBack);
```

## 开关闪光灯

### 配置步骤

1. 调用 [`isCameraTorchSupported`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/isCameraTorchSupported.html) 方法，检测设备是否支持闪光灯常开。
    ::: note note 
    - 一般情况下，App 默认开启前置摄像头，因此如果设备的前置摄像头不支持闪光灯，直接调用此方法会返回 NO（表示不支持闪光灯常开）。
    - 若您需要检查后置摄像头是否支持闪光灯，需要先调用 [`switchCamera`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/switchCamera.html) 方法切换至后置摄像头或调用 [`switchCameraWithPosition`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/switchCameraWithPosition.html) 方法指定使用后置摄像头，再调用此方法。
    :::
2. 调用 [`setCameraTorchOn`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/setCameraTorchOn.html) 开关闪光灯，通话过程中也可随时开关。


### 示例代码

```
// 检测设备是否支持闪光灯常开
await _engine.deviceManager.isCameraTorchSupported();

// 开关闪光灯
bool cameraFlashOn = true;
await _engine.deviceManager.setCameraTorchOn(cameraFlashOn);
```

## 手动对焦

### 配置步骤

1. 调用 [`isCameraFocusSupported`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/isCameraFocusSupported.html) 方法，检测设备是否支持手动对焦。
2. 调用 [`setCameraFocusPosition`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/setCameraFocusPosition.html) 方法设置手动对焦位置并触发对焦。相关参数说明如下：
    - focusX：触摸点相对于视图的横坐标，iOS取值范围为 0 ~ 1，Android取值范围为[-1100,1100]
    - focusY：触摸点相对于视图的纵坐标，iOS取值范围为 0 ~ 1，Android取值范围为[-1100,1100]
    ::: note note
    对焦点区域面积对测光和对焦都有影响，用户应根据需要调整对焦点区域面积。
    :::

### 示例代码

```
// 检测设备是否支持手动对焦
await _engine.deviceManager.isCameraFocusSupported();

// 设置手动对焦位置
await _engine.deviceManager.setCameraFocusPosition(event.getX(),event.getY());  
```

## 手动曝光

### 配置步骤

1. 调用 [`isCameraExposurePositionSupported`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/isCameraExposurePositionSupported.html) 方法，检测设备是否支持手动曝光功能。
2. 调用 [`setCameraExposurePosition`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcDeviceManager/setCameraExposurePosition.html) 方法设置手动曝光位置并触发曝光。相关参数说明如下：
    - focusX：触摸点相对于视图的横坐标，iOS取值范围为 0 ~ 1,Android取值范围为[-1100,1100]。
    - focusY：触摸点相对于视图的纵坐标，iOS取值范围为 0 ~ 1,Android取值范围为[-1100,1100]。

### 示例代码

```
// 检测设备是否支持手动曝光
await _engine.deviceManager.isCameraExposurePositionSupported();

// 设置手动曝光位置
await _engine.deviceManager.setCameraExposurePosition(event.getX(),event.getY());    
```
