NERTC SDK 支持设置音频编码属性设置，本文档为您介绍如何通过 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/setAudioProfile.html" target="_blank">`setAudioProfile`</a> 方法的 `profile` 和 `scenario` 参数实现不同的音质效果。

## <span id="功能描述">功能描述</span>

在不同的场景中，用户对于音质、声道、噪声抑制等方面的要求各有不同，例如在音乐直播、在线 KTV 等场景中，需要设置适宜于音乐场景的采样率、码率、流畅度、噪声抑制等参数和能力。NERTC 支持多种音频属性档位设置，适用于不同场景的音质、声道需求，以便您根据实际场景方便快捷地调整音质属性、在常见场景中实现最优的音质效果。

NERTC SDK 通过 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/setAudioProfile.html" target="_blank">`setAudioProfile`</a> 方法的 `profile` 和 `scenario` 参数提供多种音频属性档位和音频场景设置，您可以根据实际场景需求灵活调整音质属性。

## <span id="注意事项">注意事项</span>

- SDK 支持在加入房间通话中后，您根据实际场景变化动态调用 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/setAudioProfile.html" target="_blank">`setAudioProfile`</a> 方法。
- 若您通过 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/setChannelProfile.html" target="_blank">`setChannelProfile`</a> 接口设置房间场景为直播模式，即 `NERtcChannelProfile.liveBroadcasting`，但未调用 <a href="https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/setAudioProfile.html" target="_blank">`setAudioProfile`</a> 方法设置音频编码属性，或仅设置 `profile` 为 `profileDefault`，则 SDK 会自动设置 `profile` 为 `profileHighQuality`，且设置 `scenario` 为 `scenarioMusic`。

## 实现方法

通过 [`setAudioProfile`](https://doc.yunxin.163.com/nertc/references/flutter/dartdoc/Latest/zh/nertc/NERtcEngine/setAudioProfile.html) 设置音频属性。

### 设置音频模式

`profile` 参数代表不同的音频参数配置（音质），比如采样率、码率和编码模式，包括：

| 音频 profile 类型 | 描述 |
|---|---|
| profileDefault | 默认设置。语音场景下为 profileStandard，音乐场景下为 profileHighQuality。 |
| profileStandard | 标准音质模式。采样率为 16 kHz、语音编码、单声道、编码码率最大值为 20 Kbps。 |
| profileStandardExtend | 标准扩展模式。采样率为 16 kHz、语音编码、单声道、编码码率最大值为 32 Kbps。 |
| profileMiddleQuality| 中等音质模式。采样率为 48 kHz、音乐编码、单声道、编码码率最大值为 64 Kbps。 |
| profileMiddleQualityStereo | 中等音质模式（立体音）。采样率为 48 kHz、音乐编码、双声道、编码码率最大值为 80 Kbps。 |
| profileHighQuality | 高音质模式。采样率为 48 kHz、音乐编码、单声道、编码码率最大值为 96 Kbps。 |
| profileHighQualityStereo | 高音质模式（立体音）。采样率为 48 kHz、音乐编码、双声道、编码码率最大值为 128 Kbps。 |


### 设置音频场景

`scenario` 参数用于设置音频的使用场景，在不同场景中，声音的流畅度、噪声抑制、音质等会根据不同的场景做出优化。目前支持以下类型的场景：

| scenario 场景 | 描述 | 音量|
|---|---|
| scenarioDefault | 默认的音频应用场景（语音）。 | 通话音量。 |
| scenarioSpeech | 语音场景。推荐用于普通的音视频通话场景。对回声、噪声等环境音过滤效果较好，可以提供更为清晰的人声语音效果。 | 通话音量。|
| scenarioMusic | 音乐场景。音乐级别音质设置，高保真、超流畅，推荐用于在线 KTV、在线演唱会、音乐直播、乐器教学等高音质要求的场景。 | 媒体音量。|
| scenarioChatRoom | 语聊房场景。优先保障音频连续性与稳定性，适用于用户需要频繁上下麦的场景。仅移动端支持。 | 通话音量。|

不同的 scenario 下，设备的系统音量使用不同的音量类型。

## <span id="示例代码">示例代码</span>
```
    _engine
        .setParameters(mparams) //设置参数
        .then((value) => _engine.create(appKey: Config.APP_KEY, channelEventCallback: this, options: options))//初始化引擎
        .then((value) => _engine.setAudioProfile(mAudioProfile, mAudioScenario))//初始化后设置音频场景
        .catchError((e) {
      Fluttertoast.showToast(
          msg: 'catchError:' + e.toString(), gravity: ToastGravity.CENTER);
    }); 

```
